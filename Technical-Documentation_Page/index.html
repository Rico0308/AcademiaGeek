¿Qué es i3 WM?

i3 Window Manager es un Administrador de Ventanas de Mosaico, las plataformas de destino son los SO GNU/Linux y BSD, el código es Software Libre y de Código Abierto (FOSS) bajo la licencia BSD. i3 está dirigido principalmente a usuario y desarrolladores avanzados, sin embargo cualquier persona puede usarlo al leer la documentación de la página oficial.

Abrir Terminales

Una de las operaciones más básicas y habituales en i3 es abrir una o múltiples terminales, la combinación de teclas por defecto al instalar i3 WM para abrir terminales es $mod + Enter, siendo $mod la tecla Alt. Tras presionar esta combinación de teclas, se abrirá la terminal que está estipulada en el archivo de configuración de i3.

Cambiar el Focus en las Terminales

Al abrir múltiples terminales se requerirá cambiar entre el focus de las mismas, es por ello que la mayoría de usuarios de i3 dejan de depender del mouse y se centran mas en utilizar atajos del teclado. Para cambiar el focus entre ventanas están estipuladas por defecto las siguientes combinaciones:

  • $mod + k.
  • $mod + l.
  • $mod + alguna de las teclas de dirección.
Cambiar el Layout del Contenedor

El Layout de los Contenedores puede variar de diferentes formas según el tipo que seleccionemos, estos son:

  • Splith/Splitv: Se dimencionan para que las ventanas tengan el mismo espacio, pueden ser de forma Horizontal (Splith) o Vertical (Splitv).
  • Apilado: Solo muestra la ventana enfocada y las demás se encuentran en una lista en la parte superior apiladas de forma vertical.
  • Tabulado: Mismo principio que apilado pero la lista se encuentra de forma horizontal.
Modo Full Ventana

Para mostrar una ventada en modo Pantalla Completo basta únicamente con presionar $mod + f, en dado caso que desee salir del modo pantalla completa deberá presionar nuevamente $mod + f.

Cerrar Ventanas

Para cerrar ventanas con atajo de teclado puede optar por emplear la combinación de teclas por defecto que proporciona i3 WM, estas son $mod + Shift + q.

Reiniciar i3

Existirán casos en los cuales requiera reiniciar i3 WM por completo por diferentes razones, entre estas pueden estar los cambios en la configuración del Administrador de Ventanas, para este caso bastan con presionar $mod + Shift + r.

Modo Flotante

En ocasiones el usuario requerirá dejar de lado el modo mosaico para poder manipular a su gusto el tamaño de la ventana, para ello se emplea el modo Flotante, para alternar entre modo Mosaico a Flotante deberá presionar $mod + Shift + Espacio.

Archivo Configuración i3

La diversión y porqué no, aprendizaje de i3 se encuentra en su archivo de configuración. En este se encuentran los atajos de teclados por defecto y así mismo, se pueden añadir los que deseemos, qué aplicaciones iniciar automáticamente, cambiar colores, definir qué aplicaciones abrir en espacios de trabajos específicos, entre muchas otras cosas mas. Se debe copiar el archivo de configuración i3 que se encuentra en /etc/i3/config a ~/.i3/config o a ~/.config/i3/config; seguidamente puede empezar a editar el archivo con su editor de texto favorito.
